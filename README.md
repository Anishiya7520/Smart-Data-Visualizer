# Smart Data Visualizer

Smart Data Visualizer is a text analysis and visualization tool built using Python.

It analyzes text data and generates visual insights such as:

• Word frequency charts  
• Word cloud visualization  
• Top keywords in the dataset  

## Features

- Paste text for analysis
- Upload text files
- Automatic text preprocessing
- Interactive visualizations

## Technologies Used

Python  
Streamlit  
NLTK  
Matplotlib  
WordCloud  

## Run the Project

pip install -r requirements.txt  
streamlit run app.py
